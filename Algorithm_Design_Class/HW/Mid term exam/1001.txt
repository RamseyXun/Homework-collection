1001. 七星灯续命
Total:	445	Accepted:	100
 	 	 
Time Limit: 1sec    Memory Limit:256MB
Description
据说早在三国时期，就有伟大的丞相诸葛亮，为了延长寿命，做过七星灯续命的尝试。

而在科技发达的今天，想要延长寿命可以有多种多样的方式。在一位长者的帮助下，你弄到了若干件灯具，想要实现丞相当年未能完成的愿望。

你现在有n(1<=n<=20)样灯具，并把它们放置在一个二维平面内，这样每个灯具的位置就可以用坐标(x,y)(0<=x,y<=99)来表示。没有两个灯具拥有相同的x坐标，也没有两个灯具拥有相同的y坐标。

由于生命之光是按照x增大且y增大的方向走的，因此合理的选择灯具，作为生命之光通过的路径来说是十分重要的。现在你想知道，在合理的选择路径的情况下，生命之光最多能够通过多少个灯具（包括起终点）

 

Input
第一行为一个数T(<=20)，表示数据的组数

之后T个例子表示如下：

第一行一个整数，表示n，意义如上

之后n行，每行两个整数x,y，表示灯具的坐标

 

Output
T行，每行一个整数，表示生命之光最多能通过多少个灯具。
